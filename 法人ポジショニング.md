# 法人ポジショニング（明確化ドラフト）

90日WBS_アクションシートの M1（〜7/8, W2）の主眼は「法人ワークショップの型を作る」。
その具体タスクは**提案資料1枚**・**ワークショップ進行設計/素材**・**ターゲット法人10社リスト**。
これらを作るには、プロダクトコンセプト（＝何を測る診断か）に加えて、**法人が合理的に導入を判断できる観点**を言語化しておく必要がある。

本書はその「明確化すべき観点」を整理し、各観点を現行コンセプトに基づいてドラフト（叩き台）にしたもの。ここで固めた言葉が、提案資料1枚・ワークショップ設計・ターゲット選定の共通ソースになる。

> 位置づけ：確定仕様ではなく**ドラフト**。特に金額・工数・数値は仮置き（`（要・自社数値）`表記）。実データと意思で確定する。
> 参照：`ファネル図.md`（商品体系・設計原則）／`文脈出し分け仕様.md`（法人コピー）／`相性ルール.md`（相性の一手）／`競合ティアダウン_ABテスト案.md`（差別化）／[90日WBS・12ヶ月計画（スプレッドシート）](https://docs.google.com/spreadsheets/d/1eb-P739JJs7qi1c9Uzy_WpZhy0gsCag382YzohFOBEY/edit)。
> 本文に長音ダッシュは使わない（プロジェクトの表記ルール）。

---

## プロダクトコンセプト（前提の再確認）

- 診断は特定の場面を前提にせず、**力を抜いたときの「自然体のあなた」の対人スタイル**を映す（職場・プライベート共通）。
- 3軸（本音 O/G・衝突 B/K・重心 L/S）の組合せで**8タイプ**に確定。名乗りやすいタイプ名が共通言語になる。
- 法人の中心価値：**自律的に尊重し合えるチームワークを構築する**。
- 商品：無料診断（共通入口）→ 読み解きワークショップ（軽め・初回ファシリ後に社内自走）→ エグゼクティブマインドセッション（管理者向け1on1）。

## 明確化すべき観点（一覧）

法人の購買は「担当者の好み」でなく「組織としての合理的判断」で動く。判断に要る問いを7観点に整理した。

| # | 観点 | 答えるべき問い | なぜ法人判断に必要か |
| :-: | :-- | :-- | :-- |
| 1 | ペルソナ | 誰の課題を、誰が予算で解くのか | 買い手（意思決定者）と利用者（現場）が別。両方に効く言葉が要る |
| 2 | 解決する課題 | 何が痛いのか。放置するとどうなるか | 「あったらいい」でなく「解くべき痛み」を名指しできると予算がつく |
| 3 | 導入効果 | 導入後、何がどう変わるか。どう測るか | 支出の見返り（before→after）を具体で描けないと稟議が通らない |
| 4 | 差別化 | なぜ他手段（性格診断・研修・サーベイ・何もしない）でなくこれか | 代替と比較され「これでなければ」の理由がないと選ばれない |
| 5 | 合理的な導入理由 | なぜ投資する価値があるか。なぜ今か | 法人営業の核。低リスク・資産化・放置コスト・波及で「買う理屈」を増やす |
| 6 | ユースケース | いつ・どの場面で効くか | 「うちのこの状況で使える」と自社に接続できると導入が具体化する |
| 7 | 想定される懸念と反論 | 何が引っかかるか。どう返すか | 反論をあらかじめ潰しておくと商談が前に進む |

以下、各観点のドラフト。

## 1. ペルソナ

法人は「買い手（意思決定者）」と「利用者（現場）」が分かれる。設計原則どおり、職場改善は**トップダウン**で届ける（個人がボトムアップで職場を変える動機は弱い）。よって訴求の主対象は意思決定者、体験の主対象は現場、の二層で設計する。

### 効くターゲットの条件（軸）

白/青（ホワイト/ブルーカラー）の別ではなく、効く条件は **「関係の質がその場の成果・顧客体験に直結し、リアルタイムの相互依存が高い仕事」**。この軸で選別する。

- **◎ 強く効く**：接客・飲食・宿泊・美容・冠婚葬祭・介護・保育など。フロント/キッチン/ホール等が同時に連携し、関係の質が接客品質と離職に即はねる。
- **× 効きにくい**：ライン作業・単独作業など、工程や個人成果が支配的で人間関係の裁量が小さい仕事。
- **△ 例外的に効く**：ホワイトカラーでも少人数で密結合のチーム（制作・クリニック等）。ただしM1では後回し。

> 背景（デブリーフ仮説）：開発・営業・専門職は個人の成果にコミットする傾向が強く、関係性は後回しで「結果が出ていること」が優先されやすい。よって動機づけが弱い。まず動機が最も強いセグメントから入る。

### 優先セグメント（Tier）

| Tier | 業種 | 理由 | 人数の目安 |
| :-: | :-- | :-- | :-- |
| **Tier1（M1最速）** | 飲食・カフェ／美容室・サロン／小規模宿／**介護・保育**／**結婚式場** | オーナー・店長・施設長が即決。関係の質が接客品質と離職に直結。介護・保育は離職の代表業種で研修/補助金の原資があることも。式場は高価値かつ多チームへの横展開の伸びしろが大きい | 5〜15名（3名以下は痛みも原資も弱い） |
| **Tier2** | ホテル宴会・多店舗チェーン等 | 組織が大きく決裁者が上、商談サイクルが長い。ビーチヘッド後に横展開で狙う | チーム単位に分割 |
| **後回し** | 開発・営業・専門職中心のホワイトカラー | 個人成果志向で動機が弱い。密結合の少数チームのみ例外 | （対象外） |

### 買い手（意思決定者）

想定：**Tier1業種のオーナー・店長・施設長（＝現場の最終決裁者）**。1店舗/1施設で意思決定が完結し、トップの一存で導入が決まる層から入る（`ファネル図.md` の「意思決定者へ直接」に対応）。

| 属性 | 内容 |
| :-- | :-- |
| 役割 | オーナー／店長／施設長・園長（中小の経営者・現場責任者） |
| 評価される軸（KPI） | **離職率・定着率／採用・育成コスト**（最重要）、接客品質・顧客満足、シフトが回るか、雰囲気・クレーム |
| 抱える悩み | 「人がすぐ辞める・採用が追いつかない」「スタッフ間がギスギスして雰囲気が接客に出る」「新人が既存メンバーに馴染めない」「任せたいのに任せられない」 |
| この診断が効く点 | 摩擦を「性格の善し悪し」でなく**タイプの違い**として非人格的に言語化し、関わり方の一手まで落とす。結果として**辞めない・回るチーム**に近づける |
| 刺さるメッセージ | 「自然体で尊重し合えるチームが、辞めない・居心地のいい店をつくる」（`文脈出し分け仕様.md` の法人heroを業種向けに調整） |

### 利用者（現場＝体験の主対象）

| 属性 | 内容 |
| :-- | :-- |
| 役割 | チームメンバー全員（役職不問） |
| 動機 | 「自分のタイプを知りたい」という好奇心（負荷ゼロの9問で完了） |
| 得る価値 | 自分の強み/癖の言語化、同僚との相性、押し付けられない自己理解 |
| 波及 | 結果画面から「大切な人にも」導線で家庭・プライベートにも展開（B2C波及の種） |

### 推進者（チャンピオン）

現場のマネージャーまたはHR担当。**無料診断を自分で受けて「当たる/話せる」体感**を持つと社内推進者になる。提案は「まず意思決定者本人に1回受けてもらう」を起点にすると強い。

> ドラフト補足：初期ターゲット10社の選定基準は「①Tier1業種（接客・飲食・宿・美容・介護・保育・式場） ②5〜15名でオーナー/店長/施設長が即決 ③離職・採用難などの痛みが顕在 ④紹介経由でトップに直接届く」。この基準でリスト化する（7/8タスク：ターゲット10社リスト）。次アクションの選定マトリクス（業種 × 人数 × 決裁者 × 離職の痛み × 接点）で優先順位づけする。

## 2. 解決する課題

**一言**：職場の人間関係の摩擦は「相性が悪い」で片づけられ放置されるが、実体は**8タイプそれぞれの関わり方の癖**がすれ違いを生んでいる。癖は本人にも周りにも見えないため、我慢・誤解・抱え込みとして溜まり、ある日「離職」「不全」として表面化する。

### 現場で起きている具体パターン（8タイプの消耗ポイントに対応）

- 引っ張り役が**独走**し、周りが本音を言えず沈黙する（突撃隊長／大黒柱）。
- 正論が**逃げ場を奪い**、関係にヒビが入る（正論ハンマー）。
- 対立が**笑いや根回しで先送り**され、課題が放置される（お祭り隊長／仕掛け人）。
- 支え役が**NOを言えず抱え込み**、限界で突然プツンと切れる／良い人ほど静かに辞める（縁の下の職人／調整役）。
- 当事者意識のズレで**任せられない/任され過ぎ**が固定化する（自由人／職人）。

### この痛みが生む組織コスト（放置コスト）

| コスト | 中身 |
| :-- | :-- |
| 離職・採用/育成のやり直し | 摩擦や「静かな我慢」による退職。1名あたり採用+育成で`（要・自社数値）`規模 |
| マネジメント工数 | すれ違いの仲裁・調整に管理職の時間が溶ける |
| 意思決定の遅延 | 本音が出ず、対立が先送りされ、決まらない |
| 心理的安全性の低下 | 発言・相談が減り、生産性とイノベーションが鈍る |

> ドラフト補足：診断の各タイプは「強み」と「消耗ポイント（→セッションの橋）」を対で持つ。**課題は誰かの欠陥でなく、強みの裏面**として提示する。これが「吊し上げでなく相互理解」という体験設計の核であり、法人が安心して全社に配れる理由になる。
>
> 優先セグメント（Tier1：接客・飲食・宿・美容・介護・保育・式場）向けには、この放置コストのうち**離職・採用コスト**を主痛みに置く。関係の摩擦は接客品質と定着に即はねる業種なので、「1人辞めると採用+育成で `（要・自社数値）` が飛ぶ」を軸に、"辞めない・回るチーム"を約束する。

## 3. 導入効果

支出の見返りを before → after で描く。診断単体でなく「診断→ワークショップ→（管理職はセッション）」の一連で効果が出る設計。

| 観点 | Before（導入前） | After（導入後） |
| :-- | :-- | :-- |
| 違いの扱い | 「合う/合わない」で人を評価。摩擦が人格批判になる | 「タイプの違い」として**非人格的に**語れる共通言語ができる |
| 関わり方 | 各自が自己流。すれ違いの再生産 | 自分×相手の**相性から具体的な一手**が分かる（`相性ルール.md`） |
| チームの空気 | 押し付け or 遠慮。本音が出ない | 押し付けでなく**自律的に尊重し合う**関係 |
| マネジメント | 個人の勘で対応 | メンバーのタイプを踏まえた任せ方・声かけができる |
| 継続性 | 研修が一過性で終わる | 共通言語が社内に残り、**自走キット＋四半期再診断**で維持される |

### 効果測定（代理指標のドラフト）

「関係の質」は直接測りにくいので、代理指標で見せる。

- **短期（ワークショップ直後）**：受診率、満足度、「相手を理解できた/関わり方のヒントを得た」回答率。
- **中期（四半期）**：1on1での会話の変化、心理的安全性サーベイの簡易設問、再診断での「前回からの変化」。
- **長期**：離職率・定着率、エンゲージメントスコアの推移（`（要・自社数値／既存サーベイと接続）`）。

## 4. 差別化（なぜ他手段でなくこれか）

法人は必ず代替と比較する。主な代替は「性格診断（MBTI/16Personalities等）」「エンゲージメントサーベイ」「単発研修」「1on1ツール」「何もしない」。

| 代替手段 | できること | 足りない点 | 本製品の優位 |
| :-- | :-- | :-- | :-- |
| 性格診断（MBTI/16P等） | 自己理解・タイプ命名 | 自己理解で止まり、**職場の関わり方に接続しない** | 相性で「自分×相手の一手」まで落ちる。職場文脈の共通言語 |
| エンゲージメントサーベイ | 課題の可視化 | 課題は見えるが**打ち手がない**。スコアが下がると重い | 打ち手（関わり方・対話）まで一体。人格でなくタイプで軽く扱える |
| 単発研修 | その場の学び | **一過性**で現場に残らない | 初回ファシリ後に**社内で自走**。共通言語が資産化 |
| 1on1ツール | 面談の運用 | 「何を話すか」の中身は埋まらない | タイプ/相性が1on1の**具体的な話題と一手**を供給 |
| 何もしない | コストゼロ | 放置コスト（離職・工数・停滞）が静かに膨らむ | 無料診断で低リスクに始められる |

### 本製品固有の3点（＝コンセプト直結の差別化）

1. **「自然体のあなた」を測る**：場面ごとの振る舞いでなく素の対人スタイル。だから職場でもプライベートでも同じ言語で語れる。
2. **相性で「関わり方の一手」に落ちる**：自分×相手の3軸力学から具体的な次の一歩が出る（自己理解で終わらない）。
3. **自走化でスケール**：初回ファシリ後は社内キットで回せる。呼ぶたびに費用がかかる研修と構造が違う。

> ドラフト補足：拡散/名乗りやすさの強み（タイプ名・あるある・相性）は `競合ティアダウン_ABテスト案.md` に詳しい。法人提案では「自己理解の面白さ」より「**チームの共通言語＋自走化**」を前面に。

## 5. 合理的な導入理由（法人営業の核）

ここが今回いちばん増やしたい観点。**「買う理屈」を複数持たせる**ことで、担当者が社内を説得しやすくする。

1. **低リスクで段階導入できる**
   無料診断で全社が体験してから、有料はワークショップという軽い単位で始められる。いきなり大型投資でなく、効果を見て横展開・年間契約へ広げられる（`ファネル図.md` の導入設計に一致）。

2. **一度入れれば資産になる（ランニングでなくストック投資）**
   初回ファシリ後は社内で自走できるキット化が前提。研修のような「呼ぶたびの費用」でなく、共通言語とキットが社内に残り続ける。四半期再診断で定点観測でき、継続価値が積み上がる。

3. **放置コストを止められる**
   人間関係の摩擦は見えにくいが高コスト（離職・マネジメント工数・意思決定の遅延・心理的安全性の低下）。「何もしない」の費用を可視化し、それを止める投資として位置づける（数値は観点2の表を自社実数で）。

4. **トップダウンで一気に浸透する設計**
   意思決定者が導入を決めれば全員が同じ診断を受け、共通言語が一斉に揃う。草の根の施策と違い、組織的に効く。買い手（管理職）自身が受けると推進が加速する。

5. **従業員のwell-being／定着施策として説明できる（波及効果）**
   診断はプライベートにも使えるため、社員が家族・パートナー理解にも活用できる＝会社が渡せる福利厚生的な価値。エンゲージメント・定着の文脈で予算がつけやすい。B2C波及（結果画面の「大切な人にも」）が、法人にとっては従業員満足の付加価値になる。

6. **意思決定者自身にもリターンがある**
   エグゼクティブマインドセッションで、管理職自身の自己理解→部下との関わりの質が上がる。組織のためだけでなく、買い手個人のメリットも設計に含まれる。

> ドラフト補足：提案では 1（低リスク）→ 3（放置コスト）→ 2（資産化）の順で語ると、「軽く始められて、痛みを止め、しかも残る」という筋が通りやすい。

## 6. ユースケース（いつ効くか）

「うちのこの場面で使える」と自社に接続できると導入が具体化する。効きやすい順のドラフト。

- **新チーム立ち上げ・組織再編後**：メンバーが互いを知らない初期に共通言語を入れる。
- **新任マネージャーの着任時**：チームのタイプ地図を持って関わりを設計できる。
- **オンボーディング**：新入社員と既存メンバーの相性を早期に橋渡し。
- **1on1・評価面談の前**：何を話すか、どう伝わるかの補助線にする。
- **管理職研修/合宿**：ワークショップ本体として。以降は社内で自走。
- **四半期の定点チェック**：再診断で「前回からの変化」を見る（継続・ストック化）。

## 7. 想定される懸念と反論（先回りで潰す）

| 懸念 | 返し方（ドラフト） |
| :-- | :-- |
| 「また性格診断か。占いと何が違う？」 | 自己理解で終わらせず、**相性→関わり方の一手**まで落とす。職場の共通言語が目的で、当てること自体が目的ではない |
| 「9問で分かるのか？」 | 細かく当てるより**共通言語をすばやく揃える**のが狙い。負荷ゼロだから全員が受け、チーム全体の地図になる。深掘りはセッションで担保 |
| 「効果は続くのか？一過性では？」 | 初回ファシリ後の**自走キット＋四半期再診断**で社内に残す設計。ここが単発研修との違い |
| 「人事評価やレッテル貼りに使われないか」 | タイプは優劣でなく**強みと癖の対**。評価でなく相互理解の道具として運用ガイドで縛る |
| 「プライバシーは？」 | 個人結果の共有範囲は本人主導。全社では匿名の分布/共通言語だけを使う運用も可（`（要・運用ルール確定）`） |
| 「予算が取りにくい」 | 無料診断＋軽いワークショップで**小さく始めて実績で広げる**。well-being/定着の文脈でも起案できる |
