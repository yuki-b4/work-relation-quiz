# 紹介リード向け 深掘り読み解きメール（テンプレート）

`?ref=` 経由の紹介先が、深掘り申込で **メール＋いま引っかかっている関係** を残してくれたときに送る個別メールの雛形。
設計は第3層（LLM個別読み）に準拠：**(a)力学の見立て → (b)タイプ×状況の罠 → (c)明日の一手 → 開いたループ → セッションへの橋**。

このメールは「1通・状況反応型・セッションへ橋渡し」。3日間ガイド（`guide-3day-spec.md`）とは**出口（CTA）だけが違う**別物で、**タイプ素材とトンマナは共通のソースを使う**。

- **タイプ素材の正（ソース）は `all-types.html`**（`TYPES`＝`tsuyomi`/`aru`/`aishou`/`hitotsu`、`RADAR_META`、`SAMPLE`、`compatLine`）。3日間ガイドと同じソース。
- **トンマナ（文体）の正は `guide-3day-spec.md`** の「文体ルール」＋「編集チェックリスト（3観点）」。本書はそれを紹介リードメール用に適用する。
- 入力（回答ログから拾う）：メール ／ 自由記述（状況）／ 紹介元ref ／ タイプ ／ 5軸 ／（任意）相手タイプ
- 出力：下のメール1通（目安10〜15分で執筆）
- 本文に長音ダッシュは使わない（プロジェクトの表記ルール）。
- この雛形がそのまま M2〜M3 のLLM自動化の仕様になる（最後の節参照）。
- 関連：`guide-3day-spec.md`（3日間ガイドの生成仕様）／`week2-運用キット.md`（紹介の渡し方）／`相性ルール.md`／`ROADMAP.md`。

---

## 使い方（3ステップ）

1. 回答ログの行から `メール / 自由記述 / 紹介元ref / タイプ / 5軸（／相手タイプ）` を拾う。
2. `all-types.html` で受診タイプの `tsuyomi`・`hitotsu`・`aishou`・`SAMPLE` を確認し（相手タイプがあれば `compatLine` も）、下の **タイプ別クリブ（後半）** を見ながらテンプレの `{スロット}` を埋める。
3. 下の **トンマナ** で自己点検してから、入力されたメール宛に送る。

---

## トンマナ（`guide-3day-spec.md` と共通）

3日間ガイドと**同じ文体**で書く。読後感を揃えることで、メール → ガイド → セッションの体験が一本につながる。

**文体ルール**

- ダッシュ（— ——）禁止。「。」「、」「：」「（）」「」」でつなぐ。
- 二人称。断定しすぎず「〜がちです」「〜してみてください」。
- 必ず職場文脈（恋愛・家庭に寄せない）。
- タイプに優劣をつけない。「持ち味 → つい出る面 → 活かし方」の順で書く。
- 根は「型」と呼ぶ。性格批判にしない。

**編集チェックリスト（送信前に必ず自己点検）**

- 免責の前置き（「タイプに良い・悪いはない」等）は書かない。読み手の目的・状況から入る。
- 否定から入らない。必ず「持ち味 → だからこその落とし穴」の順。
- 助言は「減る損失」でなく「得られるもの」で締める（「消耗が減る」より「周りとの波長が合う」）。
- 行動を促すときは入口を下げる（「まずは言いやすい相手から」「一日に一つ」）。
- 指示は具体・定量に（「半歩」より「思っている半分のペース」）。一手で言える「セリフ」にできると最強。
- 主語を省かず明示する。体言止めは述語まで言い切る。

## 各スロットの書き方（ねらいと素材の出どころ）

各スロットは `all-types.html` のどのフィールドから引くかを決めておくと、3日間ガイドと素材がぶれない。

- **(a) 力学の見立て**：素材は `aishou`＋（相手タイプがあれば）`compatLine`。相手を主語にせず「関係の構造」を主語にする。「上司が安心して次を頼む流れができている」のように、誰のせいでもない仕組みとして描く。図星感はここで作る。相手タイプの推定ロジックは `guide-3day-spec.md` 5-3 に準拠。
- **(b) 罠（持ち味 → つい出る面）**：素材は `tsuyomi`＋`hitotsu`。必ず「持ち味 → だからこその落とし穴」の順。否定から入らない。クリブの言葉を相手の状況に翻訳する。
- **(c) 一手（活かし方）**：素材は `hitotsu` の核を1アクションに。1つだけにする。多いと動けない。明日その場で言える「セリフ」にできると最強。
- **開いたループ（根）**：一手で全部は解けないと正直に示す。根を「型」と呼んで、性格批判にしない。ここが高単価への橋。
- **なりたい方向（締め）**：`hitotsu` の前向きな結びから。「得られるもの」で締める。
- **CTA**：押し売りにしない。「一緒にほどく」「整理する」の協働トーン。

## タイプ別クリブ（8タイプ）

各タイプの素材。**正は `all-types.html`**（`tsuyomi`/`hitotsu`/`aishou`/`SAMPLE`）で、ここはメール用に距離を縮めた抜き書き。`(b)罠`は「持ち味＋つい出る面」、`(c)一手`は活かし方、`根`は開いたループに使う。文体は上の「トンマナ」に合わせる。

### 突撃隊長（OBL）｜気づけばいつも、先頭。
- 持ち味：停滞した空気をこじ開ける突破力。みんなが二の足を踏む場面でも、最初の一歩を踏み出して周りを巻き込んでいけます。
- つい出る面：勢いのまま話を進めてしまい、気づけば誰も本音を言っていない、ということになりがちです。強すぎる直球が、ときに相手を萎縮させます。
- 一手：進める前に、ひと呼吸おく。踏み込みたくなった瞬間に、あえて「みんなはどう思う？」と先に問いを投げてみてください。
- 誤解されやすい点：本当は成果を願って動いているのに、「独走」「自分勝手」と受け取られがちです。
- 5軸（代表例）：摩擦＝切り込む（最も強い）。本音＝率直に話す寄り。任せ方＝自分で担う寄り（抱えがち）。
- 根（型）：先に動いて場を引っ張ることで、止まる不安をしのぐ「型」。
- なりたい方向：半歩引いて、周りの一歩を引き出す。

### 正論ハンマー（OBS）｜おかしいことは、おかしい。
- 持ち味：曖昧さや欺瞞を見逃さない、筋を通す力。議論のほころびにいち早く気づき、場を整えていけます。
- つい出る面：正論を言い切った瞬間に相手が黙り込み、場の温度が下がってしまう、ということになりがちです。鋭い正しさが、ときに相手の逃げ場を奪います。
- 一手：指摘の前に、一つ受け取る。言いたくなった瞬間に、あえて「ここは良いと思う」と一つ認めてから本題に入ってみてください。
- 誤解されやすい点：本当は場を良くしたいだけなのに、「批判ばかりの人」と受け取られがちです。
- 5軸（代表例）：摩擦＝切り込む（最も強い）。間合い＝適度な距離寄り。本音＝率直に話す寄り。
- 根（型）：正しさで関係の良し悪しを測る「型」。
- なりたい方向：正しさを、信頼に変える伝え方。

### お祭り隊長（OKL）｜とにかく、盛り上げる。
- 持ち味：その場をぱっと明るくするエネルギー。誰とでも打ち解け、チームの空気そのものを作っていけます。
- つい出る面：重い話になりそうな瞬間につい冗談で流してしまい、家に帰ってから「本当は言うべきだった」と思い出す、ということになりがちです。場をしのぐ明るさが、ときに解くべき問題にフタをします。
- 一手：茶化す前に、本音を一つだけ置く。空気をゆるめたくなった瞬間に、あえて「私はこう思う」と短く本心を添えてみてください。
- 誤解されやすい点：実は繊細で嫌われるのが怖いのに、「ノリだけ」「軽い人」と受け取られがちです。
- 5軸（代表例）：摩擦＝和を保つ（最も強い）。間合い＝積極的に近づく寄り。
- 根（型）：明るさで本音にフタをして、対立を避ける「型」。
- なりたい方向：笑いに逃げず、本音も置ける場づくり。

### 自由人コメンテーター（OKS）｜言うだけ、言っておく。
- 持ち味：空気に流されない自由な視点。歯に衣着せず、ほかの人が見落とす角度から本質を突いていけます。
- つい出る面：「で、誰がやる？」という空気になった瞬間、自分だけそっと一歩引いてしまう、ということになりがちです。良い意見を言うほど、当事者になり損ねます。
- 一手：言ったことは、半歩だけ引き受ける。意見を述べた直後に、あえて「この部分なら自分がやります」と小さく手を挙げてみてください。
- 誤解されやすい点：本当は鋭く見ているのに、「評論家」「他人事」と受け取られがちです。
- 5軸（代表例）：境界＝線を引く（最も強い）。任せ方＝自分で担う寄り。摩擦＝和を保つ寄り。
- 根（型）：矢面や責任から距離を取って、身を守る「型」。
- なりたい方向：言葉に当事者性を乗せて、影響力に変える。

### 沈黙の大黒柱（GBL）｜多くは語らない。でも逃げない。
- 持ち味：危機にこそ揺るがない安定感。多くを語らずとも背中でチームを支え、いざという場面で前に出て収めていけます。
- つい出る面：本当は手一杯なのに「大丈夫」と言ってしまい、誰にも相談できないまま抱え込む、ということになりがちです。その頼もしさが、ときに「助けを求められない壁」になります。
- 一手：弱さを、先に一言だけ見せる。抱えきれないと感じた瞬間に、あえて「実はここが難しくて」と短く打ち明けてみてください。
- 誤解されやすい点：実は面倒見が良いのに、「何を考えているか分からない」「怖い」と受け取られがちです。
- 5軸（代表例）：任せ方＝自分で担う（最も強い・抱える）。本音＝胸にとどめる寄り。摩擦＝切り込む寄り。
- 根（型）：弱さを見せず、背中で示す「型」。
- なりたい方向：言葉で開いて、安心して頼られる存在に。

### 縁の下の職人（GBS）｜黙って、やり切る。
- 持ち味：黙ってやり切る信頼性。目立たなくても欠かせない土台を支え、質の高い仕事で周りに安心を届けていけます。
- つい出る面：頼まれごとを断れずに引き受け続け、ある日ふっと「もう無理だ」と一気にきてしまう、ということになりがちです。溜め込んだ我慢が、気づかないうちに負債になります。
- 一手：小さなNOを、一つ口にする。また頼まれそうになった瞬間に、あえて「今は手一杯なので、来週なら」と条件を添えて返してみてください。
- 誤解されやすい点：限界が近いのに、いつまでも「大丈夫な人」「便利な人」として扱われがちです。
- 5軸（代表例）：境界＝引き受ける（最も強い）。任せ方＝自分で担う寄り（抱える）。摩擦＝切り込む寄り。
- 根（型）：我慢を一人で抱え込む「型」。
- なりたい方向：小さなNOで、自分の容量を守る。

### 根回しの仕掛け人（GKL）｜気づいたら、私の流れ。
- 持ち味：敵を作らず物事を前に進める調整力。事前に布石を打ち、角を立てずに場を動かしていけます。
- つい出る面：根回ししておいたことが表に出て「裏で手を回した」と受け取られてしまう、ということになりがちです。「察してほしい」が強すぎると、誤解や不信の火種になります。
- 一手：要所だけは、本音を表で出す。ここぞという場面で、あえて「私はこう考えています」と一言、直接伝えてみてください。
- 誤解されやすい点：本当は平和に進めたいだけなのに、根回しがバレると「裏がある」「腹黒い」と受け取られがちです。
- 5軸（代表例）：摩擦＝和を保つ（最も強い）。本音＝胸にとどめる寄り。間合い＝積極的に近づく寄り。
- 根（型）：直接言わず、裏で流れを作って動かす「型」。
- なりたい方向：透明な相談で、信頼ごと動かす。

### がんばり屋の調整役（GKS）｜気づけば、みんなのお母さん。
- 持ち味：誰もが安心できる空気を作る気配り。対立の緩衝材となって、ぶつかりそうな場の角をやわらげていけます。
- つい出る面：みんなの機嫌に気を配って場を丸く収め、家に帰った瞬間にどっと疲れが出る、ということになりがちです。合わせすぎるうちに、自分の本音がどこかへ行ってしまいます。
- 一手：小さな「私はこう思う」を、一日に一つ。賛成でも反対でも、あえて自分の気持ちを短く言葉にしてみてください。
- 誤解されやすい点：自分を抑えているのに「いい人」として便利に頼られ、本音には気づかれにくいです。
- 5軸（代表例）：境界＝引き受ける（最も強い）。間合い＝積極的に近づく（強い）。摩擦＝和を保つ寄り。
- 根（型）：自分を後回しにして、場に合わせる「型」。
- なりたい方向：自分の機嫌を、後回しにしない関わり方。

## 自動化への布石（M2〜M3）

この雛形は、そのままLLM自動生成の仕様になる。**3日間ガイドと素材・トンマナを共有し、違うのは出口（CTA）だけ**にする。

- **入力**：タイプ（コード/名前）／5軸スコア／自由記述（状況）／紹介者名／（任意）相手タイプ
- **参照知識（正）**：`all-types.html`（`TYPES`/`RADAR_META`/`SAMPLE`/`compatLine`）。3日間ガイドと同一ソース。
